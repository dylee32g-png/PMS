## 구현 예정 기능 (항목 그림 → 코드 분리 후)
1. **공사계약·공사완료** → 날짜 칸(달력 입력, 빈칸도 입력 가능)
2. **날짜↔내용 자동 연동** → 내용 수정 시 날짜를 오늘로 자동 갱신
3. **관리자** 컬럼 신규 추가(선택적, 없을 수 있음)
4. **발주처(③ U열) → '발주처 담당자'로 이름 변경**
5. **O 체크 항목 3개**(영업견적·공사계획서·완료처리) 클릭 토글
6. **참조 = 폴더명 표시 + UNC 경로 복사 버튼**
7. **표 항목 숨김/표현 기능** — 대상: 도면입수·I/O Map·화면작성·기준정보생성·PLC·ETOS·HMI·자체시운전·통합시운전·총진척률(10개) + ④안전관리 8개
8. **진행현황 Hold/HOLD 통일**
9. **표 ↔ 상세/수정 팝업 동기화**(기본 컨셉, 팝업 복사본 빈틈 보완 포함)
